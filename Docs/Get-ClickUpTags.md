# Get-ClickUpTags

## SYNOPSIS
Get all ClickUp space tags.

## SYNTAX

```
Get-ClickUpTags [-SpaceID] <UInt32> [<CommonParameters>]
```

## DESCRIPTION
Get all ClickUp space tags.

## EXAMPLES

### EXAMPLE 1
```
Get-ClickUpTags -SpaceID 512
Get all ClickUp tags for space with ID "512".
```
