# Get-ClickUpFolder

## SYNOPSIS
Get properties on a particular ClickUp folder.

## SYNTAX

```
Get-ClickUpFolder [-FolderID] <UInt64> [<CommonParameters>]
```

## DESCRIPTION
Get properties on a particular ClickUp folder.

## EXAMPLES

### EXAMPLE 1
```
Get-ClickUpFolder -FolderID 11111111
Get ClickUp folder with ID "11111111".
```
