# Add-ClickUpAPIKey

## SYNOPSIS
Add a ClickUp API key to the current session.

## SYNTAX

```
Add-ClickUpAPIKey [-APIKey] <String> [<CommonParameters>]
```

## DESCRIPTION
Add a ClickUp API key to the current session.

## EXAMPLES

### EXAMPLE 1
```
Add-ClickUpAPIKey -APIKey pk_11111111_AAAAAAAAAAAAAAAAAAA1111111111111
Add the ClickUp API key "pk_11111111_AAAAAAAAAAAAAAAAAAA1111111111111" to the current session.
```
